Lecture 3: Logistic Regression + Cross-Entropy

We use logistic regression: \[s = w^T x + b, \qquad z = \sigma(s)\]

Given: \[w = [1,-2], \qquad b = 0.5, \qquad x = [2,1]\]

Tasks:

  1. Compute the logit: \(s = w^T x + b\).

  2. Compute the predicted probability: \(z = \sigma(s)\).

  3. Compute the cross-entropy loss for [make sure to use log base 2!]:

    • \(y=1\): \(L = -\log_2(z)\)

    • \(y=0\): \(L = -\log_2(1-z)\)

Round all final answers to 2 decimals.
