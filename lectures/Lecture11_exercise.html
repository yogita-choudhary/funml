Lecture 11: Clustering (k-Means)

In k-Means, we assign each point to its closest centroid (Euclidean distance). Equivalently, we can compare squared distances (to avoid square roots): \[d(\bm{x},\bm{m}_j)^2 = (x_1-m_{j1})^2 + (x_2-m_{j2})^2, \qquad \text{Cluster}(\bm{x})=\arg\min_j d(\bm{x},\bm{m}_j)^2.\] Tie-breaking rule: if there is a tie, choose the cluster with the smallest index.

Given centroids: \[\bm{m}_1 = (2,3),\quad \bm{m}_2 = (7,8),\quad \bm{m}_3=(5,2)\] New data point: \[\bm{x}_{\text{new}} = (4,4)\]

Questions (multiple choice):

  1. Which statement is correct?

    • Minimizing \(d(\bm{x},\bm{m}_j)\) is different from minimizing \(d(\bm{x},\bm{m}_j)^2\).

    • Minimizing \(d(\bm{x},\bm{m}_j)\) is equivalent to minimizing \(d(\bm{x},\bm{m}_j)^2\).

    • Squaring the distance changes the nearest centroid whenever distances are \(<1\).
