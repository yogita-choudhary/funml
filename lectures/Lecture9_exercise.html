Lecture 9: Regularization and Performance Metrics

This exercise practices both computing and interpreting regression metrics and ridge regularization.

Given predictions and targets \[\bm{y} = \begin{bmatrix} 1 \\ 2 \\ 2 \end{bmatrix}, \qquad \hat{\bm{y}} = \begin{bmatrix} 1 \\ 1 \\ 3 \end{bmatrix}, \qquad N=3\]

A trained model has parameters \[\bm{\theta}= \begin{bmatrix} 2 \\ -1 \end{bmatrix}, \qquad \|\bm{\theta}\|_2^2 = 5, \qquad \gamma=0.60\]

Ridge objective: \[L(\bm{\theta}) = \underbrace{\frac{1}{N}\sum_{i=1}^N(\hat{y}_i-y_i)^2}_{\text{data fit (MSE)}} + \underbrace{\frac{\gamma}{N}\|\bm{\theta}\|_2^2}_{\text{regularization}}\]

Select the best answer for each question.

  1. What is the residual vector \(\bm{r}=\hat{\bm{y}}-\bm{y}\)?

    • \(\begin{bmatrix}0\\1\\1\end{bmatrix}\)

    • \(\begin{bmatrix}0\\-1\\1\end{bmatrix}\)

    • \(\begin{bmatrix}1\\-1\\1\end{bmatrix}\)

  2. What is the Mean Squared Error (MSE)?

    • \(0.33\)

    • \(0.67\)

    • \(1.00\)

  3. What is the ridge objective value \(L(\bm{\theta})\)?

    • \(0.67\)

    • \(1.00\)

    • \(1.67\)

  4. Suppose the third prediction changes from \(3\) to \(8\). Which metric would increase the most?

    • MAE

    • RMSE

    • MSE

  5. Two models have the same MSE, but Model A has much larger \(\|\bm{\theta}\|_2^2\). Which model does ridge regression prefer?

    • Model A (larger weights)

    • Model B (smaller weights)

    • Both equally

  6. If \(\gamma\) increases, what happens to the learned weights?

    • Weights grow larger

    • Weights shrink toward zero

    • Weights stay unchanged
