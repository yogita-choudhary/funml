Lecture 17: Best Practices

  1. Loss for balanced 10-class classification
    Choose the best loss function.

    1. Mean Squared Error (MSE)

    2. Cross-Entropy

    3. L1 Loss

    4. Weighted Cross-Entropy

    Solution 1:
    For balanced multi-class classification, cross-entropy is the standard choice because it directly models class probabilities and works well when classes are evenly represented. \[\boxed{\textbf{Answer: (B)}}\]

  2. Loss for imbalanced tumor segmentation
    Choose the best loss function.

    1. Cross-Entropy

    2. Weighted Cross-Entropy

    3. MSE

    4. Focal Loss

    Solution 2:
    Focal loss emphasizes difficult and minority examples, making it particularly effective for highly imbalanced segmentation problems. \[\boxed{\textbf{Answer: (D)}}\]

  3. Fix exploding gradients via activation histogram
    What is the best fix?

    1. Increase learning rate

    2. Decrease learning rate

    3. Use a more aggressive activation like ReLU

    Solution 3:
    Exploding gradients indicate updates are too large; reducing the learning rate stabilizes training. \[\boxed{\textbf{Answer: (B)}}\]
