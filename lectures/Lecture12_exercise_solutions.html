Lecture 12: GMMs and Performance Evaluation

Given table:

Sample Features GT Cluster Method 1 Method 2
1 [0,1] 1 1 1
2 [1,1] 1 1 2
3 [3,1] 2 2 2
4 [5,1] 2 3 3
5 [6,1] 3 3 3

  1. Question: Which Rand Index values are correct?

    • \(RI_1=0.60,\; RI_2=0.80\)

    • \(RI_1=0.80,\; RI_2=0.60\)

    • \(RI_1=1.00,\; RI_2=0.80\)

    Solution:

    There are \(\binom{5}{2}=10\) total pairs.

    Method 1: \[RI_1=\frac{1+7}{10}=0.80\]

    Method 2: \[RI_2=\frac{0+6}{10}=0.60\]

    \[\boxed{(B)}\]

  2. Question: Which Purity values are correct?

    • \(P_1=0.60,\; P_2=0.80\)

    • \(P_1=0.80,\; P_2=0.60\)

    • \(P_1=1.00,\; P_2=0.80\)

    Solution:

    Method 1 clusters: \[P_1=\frac{2+1+1}{5}=0.80\]

    Method 2 clusters: \[P_2=\frac{1+1+1}{5}=0.60\]

    \[\boxed{(B)}\]

  3. Question: If ground-truth labels are unknown, which metric is most appropriate?

    • Rand Index

    • Silhouette Coefficient

    • Purity

    Solution:

    Silhouette is an internal metric that does not require labels.

    \[\boxed{(B)}\]

  4. Question: Which statement describes a limitation of Purity?

    • It penalizes having too many clusters.

    • It can increase artificially when the number of clusters increases.

    • It does not require ground truth labels.

    Solution:

    Purity can be maximized by assigning each point its own cluster.

    \[\boxed{(B)}\]

  5. Question: What does a high silhouette score indicate?

    • Clusters are overlapping and poorly separated.

    • Points are close to their own cluster and far from other clusters.

    • The number of clusters equals the number of data points.

    Solution:

    High silhouette \(\Rightarrow\) good separation and compact clusters.

    \[\boxed{(B)}\]
