Lecture 12: GMMs and Performance Evaluation

We will evaluate clustering outputs using performance metrics.

Given: The following clustering results from two methods on five samples, along with ground truth (GT):

Sample Features GT Cluster Method 1 Method 2
1 [0,1] 1 1 1
2 [1,1] 1 1 2
3 [3,1] 2 2 2
4 [5,1] 2 3 3
5 [6,1] 3 3 3

Round numerical answers to two decimals.

  1. Which Rand Index values are correct?

    • \(RI_1=0.60,\; RI_2=0.80\)

    • \(RI_1=0.80,\; RI_2=0.60\)

    • \(RI_1=1.00,\; RI_2=0.80\)

  2. Which Purity values are correct?

    • \(P_1=0.60,\; P_2=0.80\)

    • \(P_1=0.80,\; P_2=0.60\)

    • \(P_1=1.00,\; P_2=0.80\)

  3. If ground-truth labels are unknown, which metric is most appropriate?

    • Rand Index

    • Silhouette Coefficient

    • Purity

  4. Which statement describes a limitation of Purity?

    • It penalizes having too many clusters.

    • It can increase artificially when the number of clusters increases.

    • It does not require ground truth labels.

  5. What does a high silhouette score indicate?

    • Clusters are overlapping and poorly separated.

    • Points are close to their own cluster and far from other clusters.

    • The number of clusters equals the number of data points.
