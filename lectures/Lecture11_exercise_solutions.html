Lecture 11: Clustering (k-Means)

Given centroids: \[\bm{m}_1 = (2,3),\quad \bm{m}_2 = (7,8),\quad \bm{m}_3=(5,2)\] New data point: \[\bm{x}_{\text{new}} = (4,4)\] Tie-breaking rule: if there is a tie, choose the cluster with the smallest index.

  1. Question: Which statement is correct?

    • Minimizing \(d(\bm{x},\bm{m}_j)\) is different from minimizing \(d(\bm{x},\bm{m}_j)^2\).

    • Minimizing \(d(\bm{x},\bm{m}_j)\) is equivalent to minimizing \(d(\bm{x},\bm{m}_j)^2\).

    • Squaring the distance changes the nearest centroid whenever distances are \(<1\).

    Solution: Since squaring is a strictly increasing function on nonnegative numbers, the \(\arg\min\) is unchanged. \[\boxed{(B)}\]

  2. Question: Compute squared distances. Which option matches \((d_1^2,d_2^2,d_3^2)\)?

    • \((5,\;25,\;5)\)

    • \((25,\;5,\;5)\)

    • \((5,\;20,\;4)\)

    Solution: \[d_1^2=(4-2)^2+(4-3)^2=4+1=5\] \[d_2^2=(4-7)^2+(4-8)^2=9+16=25\] \[d_3^2=(4-5)^2+(4-2)^2=1+4=5\] \[\boxed{(A)}\]

  3. Question: Which cluster should \(\bm{x}_{\text{new}}\) be assigned to?

    • Cluster 1

    • Cluster 2

    • Cluster 3

    Solution: We have a tie: \(d_1^2=d_3^2=5\). Tie-breaking picks the smallest index. \[\boxed{(A)}\]

  4. Question: If Cluster 1 contains \(\bm{x}^{(1)}=(2,3)\) and \(\bm{x}^{(2)}=(4,4)\), what is the updated centroid \(\bm{m}_1\)?

    • \((3,3.5)\)

    • \((3,4)\)

    • \((4,3.5)\)

    Solution: Centroid is the mean of coordinates: \[\bm{m}_1=\left(\frac{2+4}{2},\frac{3+4}{2}\right)=(3,3.5)\] \[\boxed{(A)}\]

  5. Question: Which condition indicates k-Means has converged?

    • Centroids stop changing (or change is negligible), so assignments also stop changing.

    • Centroids keep changing but assignments stop changing.

    • Assignments keep changing but centroids stop changing.

    Solution: Convergence is reached when the algorithm stabilizes (no meaningful change in centroids / no reassignment). \[\boxed{(A)}\]

  6. Question: Which statement is true about k-Means?

    • k-Means always finds the global optimum regardless of initialization.

    • k-Means can converge to a poor local solution depending on initialization.

    • k-Means is not affected by outliers because it uses Euclidean distance.

    Solution: k-Means depends on initialization and can get stuck in local minima; it is also sensitive to outliers. \[\boxed{(B)}\]
