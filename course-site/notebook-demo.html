Welcome to this demo page showcasing various ways to embed and share interactive computational notebooks. Whether you're working with Python, Julia, R, or Wolfram Language, there are multiple platforms that allow you to share your work in an accessible and interactive format.

Below you'll find examples of different embedding methods, each with its own strengths. Some provide read-only views perfect for documentation, while others offer full interactivity where visitors can modify and run code directly in their browser.

üìä Google Colab Notebook

Google Colab is ideal for Python-based machine learning and data science workflows. It provides free GPU/TPU access and integrates seamlessly with Google Drive.

Note: Replace the URL with your own: https://colab.research.google.com/github/YOUR-USERNAME/YOUR-REPO/blob/main/notebook.ipynb

üìì Jupyter Notebook (nbviewer)

nbviewer provides a clean, static rendering of Jupyter notebooks. Perfect for sharing analyses, tutorials, or documentation without requiring users to run code.

Note: Use this format: https://nbviewer.org/github/YOUR-USERNAME/YOUR-REPO/blob/main/notebook.ipynb

üî¨ Interactive Binder Environment

Binder creates a fully interactive computing environment in the cloud. Users can modify code, install packages, and experiment freely. Supports Python, R, Julia, and more.

Note: Format: https://mybinder.org/v2/gh/YOUR-USERNAME/YOUR-REPO/main?filepath=notebook.ipynb
First launch takes 30-60 seconds to build the environment.

üßÆ Wolfram Cloud Notebook

Wolfram notebooks bring symbolic computation, visualization, and the Wolfram Language to the web. Great for mathematical modeling, symbolic math, and computational essays.

To embed a Wolfram notebook:
  1. Publish your notebook to Wolfram Cloud
  2. Get the cloud object URL
  3. Use this format: <iframe src="https://www.wolframcloud.com/obj/YOUR-CLOUD-PATH" width="100%" height="700px"></iframe>
